## 🚀 STARTING YOUR SYSTEM

### Option 1: Automated Setup (Recommended)

**Linux/Mac:**
```bash
chmod +x setup.sh
./setup.sh
```

**Windows:**
```bash
setup.bat
```

### Option 2: Manual Setup

```bash
# Install dependencies
npm install

# Create environment file
cp .env.example .env

# Edit .env with your MongoDB URI
# nano .env (or use your editor)

# Start the server
npm start
```

## 🌐 API ENDPOINTS REFERENCE

### Movies
| Method | Endpoint | Purpose |
|--------|----------|---------|
| GET | `/api/movies` | Get all movies |
| GET | `/api/movies/:id` | Get single movie |
| POST | `/api/movies/save` | Save movie |
| PUT | `/api/movies/:id` | Update movie |
| DELETE | `/api/movies/:id` | Delete movie |
| GET | `/api/movies/search?query=X` | Search movies |

### Admin
| Method | Endpoint | Purpose |
|--------|----------|---------|
| POST | `/api/scrape` | Scrape URL |
| GET | `/api/stats` | Get stats |

## 📚 USAGE WORKFLOW

### Typical Admin Workflow
```
1. Access Admin Panel → Tab "⚙️ Admin"
2. Paste Movie URL → Field "Movie/Show URL"
3. Click "🔍 Fetch & Preview" → Scraper extracts data
4. Review Preview → Check title, poster, video
5. Click "✅ Confirm & Save" → Saves to database
6. Movie appears on homepage → Available to users
```

### Typical User Workflow
```
1. Open Homepage → Grid of movies loads
2. Hover over Movie → Play button appears
3. Click Movie → Video modal opens
4. Watch Video → Full-screen available
5. Close Video → Back to grid
```
